Minimum Contrast Ratio

According to the SAP Product Standard, text and icons need to fulfill the following contrast requirements:

4.5:1

Text,
text-like icons

3:1

Headings, large text
(18 pt regular or 14 pt bold)

2:1

Area Icons

No special contrast ratio is required for

  • Disabled text
  • Incidental (i.e. purely decorative) elements
  • Logos and brand names

How to measure contrast ratios correctly

Because of Microsofts Cleartype mechanism the character pixels of a font might contain several color values to improve visibility. In order to measure the correct contrast value you need to measure pixels at the extreme ends of the color spectrum. For example, if you have dark characters on a light background you need to take the darkest pixels for the measurement. Vice versa if you have light characters on a dark background, you nee to take the lightest pixels for the measurement.
