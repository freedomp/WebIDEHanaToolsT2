• SAP Web IDE — Design Guidelines

Visual Design Foundations

Icons

Icons are rare in the SAP Web IDE. This intended absence of visual elements opposes the status quo in integrated development environments that are often crowded with visual information. Thus, reducing the amount of icons to a minimum, is an important factor for its contemporary and clutter-free visual design.

Currently icons occur only in these functional areas:

Toolbar (16 pixels, 1 rem)


Sidebars (16 pixels, 1 rem)

Wizard tiles (32 pixels, 2 rem)

Repository browser tree (16 pixels, 1 rem)

Control Pallet of the WYSIWYG editor (32 pixels, 2 rem) Show all

For its (rare) icons the SAP Web IDE makes use of two icon fonts. The SAPUI5 icon font which is included in Fiori, as well as its own set of icons, the SAP Web IDE icon font. The shapes of the SAP Web IDE icon font are slightly slimmer and even more simplified, but both icon fonts can be displayed side by side due to their similar styles.

Each of their icons has been hand crafted, hand hinted, and aligned, using metaphors that have been tailored for simple and direct user interaction. The hinting process ensures that whether the icons are displayed on high or low resolution screens, they are always crisp and clear.

The SAPUI5 icon font consists of more than 500 scalable generic pictograms in SAP's illustrative style. It should be used for icons in the plug-in toolbar and sidebars, and for tiles in the wizard.

The SAP Web IDE icon font is adapted to the specific use cases of an IDE. It currently contains more than 30 special icons. This icon font should be used for the toolbar, the side bars, and the WYSIWYG editor.

Drawing Icons

In case there are no suitable icons for your plugin in both fonts, we either draw the icons for you or you can do it yourself. Please get in touch with us so that we can keep the icon stylings aligned.

  • Icons in the flat style usually depict only one item
  • Items are shown from their most characteristic side (top or front) and perspective usually is avoided
  • Plain items work best
  • Lines in icons are exceptionally slim (two pixels maximum)
  • Icons usually do not contain type or numbers
  • Icons in the SAP Web IDE are monochromatic
  • We recommend Font Lab Studio to draw and hint icons
  • Clear type rendering requires designers to align their shapes to the pixel grid, it also favors horizontal and vertical lines, as well as lines with 45 degrees

Icons with States

To indicate states and statuses icons in the SAP Web IDE can either be colored or a decorator can be placed in its bottom left corner. Changes to the file itself (e.g. making it public) are represented by a color change to the icon itself while errors and similar state changes in the file are shown as decorators.

Icons with different states in the repository browser tree (the green file is public, icons with a decorators contain errors and warnings).

Do

  • Re-use existing icons
  • Hint your icons for a font-size of 16 pixels (1 rem)
  • In case there are two similar icons in both fonts, use the SAP Web IDE icon font

Don't

  • Do not submit changes to the icon font in the current implementation without notifying us!
  • Do not add new actions to the toolbar but rather to the menu (only the most frequently used actions are supposed to go into the toolbar)
