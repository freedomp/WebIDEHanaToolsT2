UI Controls

Buttons

The button control represents a simple push button. It is used for initiating actions, such as save or print. It can contain only text, no icons; the order of the two can be configured. By applying the additional style class .sapWattControlSmall you can switch to a small appearance.

There are several button types for the SAP Web IDE:

  • Normal Buttons
  • Small Buttons
  • Large Buttons

Normal Size

The normal buttons are based on a height of 2 rem / 32 pixels. They should be used as a default throughout the SAP Web IDE, i.e. main editor area, overlays, pop-ups, the wizard etc.
