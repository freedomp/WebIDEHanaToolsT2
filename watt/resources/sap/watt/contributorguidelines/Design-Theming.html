Visual Design Foundations

Theming

Currently the code editor of the SAP Web IDE come with four themes: Cumulus, Basement, Morlock, and Flower Power.

We plan to extend its theming capabilities from the editor to all of the user interface elements of the SAP Web IDE. Because of this direction it is important to use Less parameters for colors where possible already now. All parameters concerning the visual styling of your plugin are clearly structured and documented in the SAP Web IDE wiki.

Do

  • Use Less parameters for colors where available
  • If not available for your plugin, stick with the color values provided in the wiki page
  • In case you require a special color, derive it from an existing one by lightening or darkening it

Don't

  • Do not create custom colors for plug-ins
  • Avoid hard coding of color values
