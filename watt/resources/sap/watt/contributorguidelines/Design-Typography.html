• SAP Web IDE — Design Guidelines

Visual Design Foundations

Typography

SAP Web IDE uses a set of timeless, sans-serif system fonts. To ensure that the design works well across all browsers, devices, and platforms, three font stacks are used within the CSS.

General Text

Usually the typography of SAP Web IDE makes uses of Arial, which falls back to sans-serif, in case it is not available.
The sizes for general texts are

  • 10 pixels for small texts and the small control set,
  • 12 pixels for the majority of texts, and
  • 14 pixels for bigger reading texts as it occurs in the large control set, wizards, overlays and dialogs.

 

Special Font for Headings and Accents

To give SAP Web IDE a modern and lightweight appearance, the comprehensive use of Arial is supplemented by an uncommonly big and very thin typeface. It serves as a counterbalance and is used for headings and special text only. The font stack uses Segoe UI Light on Windows and Neue Helvetica on Macintosh; a Lineal Humanist or a very thin Grotesque font, both very modern genres of sans-serif typefaces.
The sizes for special text are

  • 48 pixels for large headings,
  • 28 pixels for sub-headings, and
  • 19 pixels for small and panel headings.

Monospaced Font

SAP Web IDE makes use of a real classic from the visual heritage of SAP’s design history. SAP Web IDE additionally uses “Arial Monospaced for SAP” for code in the editor and also to visually differentiate code from normal text in paragraphs in lists.
The sizes for mono-spaced text are

  • inherited from the containing in-line text, and are
  • 12 pixels in the editor.

 

System Fonts?

System fonts offer the advantage of no loading time, ensuring that there is always a consistent font available that works with the minimalistic and reduced style that comes with SAP Web IDE. The used font stack comes with a full Unicode solution when special characters are needed. In addition to this, the SAP Web IDE is proud to be web-based; this it quotes from the visual heritage of the World Wide Web, where Arial is a real classic—used and frequently tested—all over the world every.
